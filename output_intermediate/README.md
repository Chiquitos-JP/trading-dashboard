# output_intermediate - 中間生成物

このディレクトリには、分析処理の中間段階で生成されるHTMLチャートやCSVファイルが保存されます。

## ディレクトリ構成

- `market/`: マーケットデータの可視化（AD Lineチャートなど）
- `macro/`: マクロデータの可視化（ダッシュボード、カードなど）
- `economicCalendar/`: 経済指標カレンダーの可視化（HTMLカレンダーなど）

## データフロー

```
data/ (生データ)
    ↓
[分析処理]
    ↓
output_intermediate/ (中間生成物) ← ここ
    ↓
[Quartoレンダリング等]
    ↓
output/ (最終成果物)
```

## 注意事項

- このディレクトリのファイルは、Quartoレンダリングや最終レポート生成の際に参照されます
- 手動で削除する場合は、再生成が必要です

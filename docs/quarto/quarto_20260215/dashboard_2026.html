2026 Trading Dashboard

Financial Statements — FY2026

Year-to-Date Performance & Financial Position

Investment Performance Dashboard • Updated:
Abstract

本ダッシュボードは2026年度の取引パフォーマンスを財務諸表形式で表示したものである。 月次KPI（Table 1）に加え、損益計算書 (P/L: Table 2)、貸借対照表 (BS: Table 3)、 キャッシュフロー計算書 (CF: Table 4)、口座別リスク分析 (Table 5) を通じて、 投資活動の財務的影響を定量的に把握する。 なお、本年度においては取引活動以外の収益・投資は存在しない。

Contents
  • Table 1. Monthly KPI Summary (業績指標)
  • Table 2. Income Statement (損益計算書 / P/L)
  • Table 3. Balance Sheet (貸借対照表 / BS)
  • Table 4. Cash Flow Statement (CF計算書)
  • Table 5. Cash Reserve Ratio (口座別リスク分析)
⁂

[1] 楽天証券およびSBI証券のデータでは諸経費は差し引かれているが、税金は含まれていない。

[2] 適用為替レートはFRED (Federal Reserve Economic Data) の日次データを使用。 DEXJPUS Series

[3] BS・CFは取引損益の累積のみを反映。期首の元本・借入等は含まない。

[4] 本レポートは run_all.py により自動生成される。

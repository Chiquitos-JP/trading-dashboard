Interactive Dashboard

Python Panelを使用した高機能分析環境です。リアルタイムフィルタリングやチャートのドリルダウンなど、自由度の高いデータ探索が可能です。

  • 動的なデータフィルタリング
  • 詳細なドリルダウン分析
  • リアルタイム描画

Always Available

Static Report

GitHub Pages上で常時閲覧可能な静的レポートです。年度ごとのKPIや月次サマリーを表形式で素早く確認できます。

  • ブラウザから即座にアクセス
  • 年度別KPIサマリー表
  • サーバー起動不要

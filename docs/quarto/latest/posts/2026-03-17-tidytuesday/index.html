Overview

When someone says a risk is “likely” or “probable,” what probability do they actually mean? This week’s TidyTuesday uses the CAPphrase dataset – over 98,000 probability judgements from 5,000+ participants – to show that verbal probability is not a stable interface.

The same word can mean 60% to one person and 90% to another. In risk communication, finance, and forecasting, this ambiguity creates hidden decision risk.

  • Data Source: CAPphrase via TidyTuesday 2026-03-10
  • Participants: 5,174 respondents
  • Phrases: 19 common probability expressions
  • Visualization: ggplot2 + patchwork (static panels)

Visualizations

1. Phrase Ranking: Median Probability with Spread

How do people map words to numbers? Each dot shows the median estimate; the bar shows the interquartile range (25th-75th percentile).

Show code 
phrase_stats <- judgements |>
  group_by(term) |>
  summarise(
    median_prob = median(probability, na.rm = TRUE),
    mean_prob = mean(probability, na.rm = TRUE),
    sd_prob = sd(probability, na.rm = TRUE),
    q25 = quantile(probability, 0.25, na.rm = TRUE),
    q75 = quantile(probability, 0.75, na.rm = TRUE),
    q10 = quantile(probability, 0.10, na.rm = TRUE),
    q90 = quantile(probability, 0.90, na.rm = TRUE),
    n = n(),
    .groups = "drop"
  ) |>
  mutate(iqr = q75 - q25) |>
  arrange(median_prob)

phrase_stats$term <- factor(phrase_stats$term, levels = phrase_stats$term)

p1 <- ggplot(phrase_stats, aes(y = term)) +
  geom_segment(aes(x = q10, xend = q90, yend = term),
               color = "#334155", linewidth = 1.5) +
  geom_segment(aes(x = q25, xend = q75, yend = term),
               color = "#3b82f6", linewidth = 3, alpha = 0.8) +
  geom_point(aes(x = median_prob), color = "#f8fafc", size = 3) +
  geom_vline(xintercept = 50, linetype = "dashed", color = "#64748b", linewidth = 0.5) +
  scale_x_continuous(
    breaks = seq(0, 100, 25),
    labels = \(x) paste0(x, "%"),
    limits = c(0, 100)
  ) +
  labs(
    title = "How People Interpret Probability Phrases",
    subtitle = "Dot = median | Thick bar = IQR (25-75%) | Thin line = 10-90%",
    x = "Probability (%)",
    y = NULL
  ) +
  theme_cap

3. Round-Number Bias: Do People Think in 5s and 10s?

Show code 
rounding <- judgements |>
  mutate(
    ends_0 = probability %% 10 == 0,
    ends_5 = probability %% 5 == 0 & !ends_0,
    round_type = case_when(
      ends_0 ~ "Multiple of 10",
      ends_5 ~ "Multiple of 5",
      TRUE ~ "Other"
    )
  ) |>
  count(round_type) |>
  mutate(pct = n / sum(n))

round_colors <- c("Multiple of 10" = "#3b82f6", "Multiple of 5" = "#f59e0b", "Other" = "#475569")

p3 <- ggplot(rounding, aes(x = round_type, y = pct, fill = round_type)) +
  geom_col(width = 0.6) +
  geom_text(aes(label = percent(pct, accuracy = 0.1)),
            vjust = -0.5, color = "#e2e8f0", size = 4) +
  scale_y_continuous(labels = percent, limits = c(0, max(rounding$pct) * 1.15)) +
  scale_fill_manual(values = round_colors, guide = "none") +
  labs(
    title = "Round-Number Bias",
    subtitle = "People overwhelmingly estimate in multiples of 5 or 10",
    x = NULL,
    y = "Share of Responses"
  ) +
  theme_cap

Key Findings

  1. “Likely” means anywhere from 60% to 90% depending on who you ask – its IQR spans ~20 percentage points
  2. Mid-range phrases are the most dangerous: terms near 50% (e.g., “possible,” “realistic possibility”) show the highest interpretation variance
  3. Round-number bias is extreme: the majority of estimates fall on multiples of 5 or 10, suggesting people approximate rather than calibrate
  4. Implication for risk communication: using verbal probability in reports, forecasts, or trading calls introduces hidden noise that numerical expressions would eliminate

4. **Implication for risk communication**: using verbal probability in reports, forecasts, or trading calls introduces hidden noise that numerical expressions would eliminate
